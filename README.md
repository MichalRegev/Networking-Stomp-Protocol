# Networking-Stomp-Protocol
The ptoject include implement of both a server,which provides STOMP server services
and a client, which a user can use in order to interact with the rest of the users.
The server will be implemented in Java and will support both Thread-Per-Client (TPC) and the Reactor.
The client implemented in C++.

Description of the project:
The world cup is upon us, and you want to stay updated.
Thus, in this project is implemented a ”community-led” world cup update subscription service.
Users can subscribe to a game channel and report and receive reports
about the game to and from the other subscribed users.
